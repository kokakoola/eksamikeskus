WideArea?

WideArea is simple and lightweight JavaScript and CSS library (2KB JS and 4KB CSS) which helps you to write better, simpler and faster.

How to use?

Really simple. Just include both JavaScript and CSS file and then add data-widearea="enable" attribute to your textarea and then call this JavaScript function in the page: wideArea();
